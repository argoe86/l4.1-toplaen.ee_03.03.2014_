Kataloogis "x-digidoc plugin" on uue Id-kaardi baastarkvara käigus loodud veebis allkirjastamise veebiplugina API dokumentatsioon ja lihtne kasutusnäidis.

--

Kataloogis "javascript teek" on kirjeldatud veebis allkirjastamise rakendustest kasutatav javascript teek, mis võimaldab nii uue kui vana Id-kaardi baastarkvara kasutajatel allkirjastamise veebirakendust kasutada. Erinevate brauserite ja operatsioonisüsteemide kombinatsioonide jaoks on loodud mitmeid veebis allkirjastamise pluginaid - antud teek peidabki allkirjastamise rakenduse looja eest erinevate pluginate API'de erinevused ära ning laadib veebilehele plugina vastavalt kasutaja arvuti operatsioonisüsteemile, brauserile ja pluginale, mis kasutaja arvutis olemas on.

* idCard.js sisaldabki mainitud javascripti teeki,
* sign.html on primitiivne näidis ja kirjeldus, kuidas veebis allkirjastamise protsess välja näeb.
* "Veebis signeerimise Javascripti klienditeek v 0.x.doc" kirjeldab javascripti teegi API't.
